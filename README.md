# NextExplorer Enterprise Manager

PowerShell-based deployment and management tool for **NextExplorer** on **Windows 11** using **Docker Desktop**.

This script helps you configure storage paths, manage server IP and port, start and stop the application, and perform cleanup or full uninstall operations.

---

## Badges

- Windows 11  
- PowerShell  
- Docker Desktop  
- External HDD  
- NAS Share  
- LAN File Sharing  
- Menu Driven  

---

# Overview

This project creates a **browser-based file portal** using **NextExplorer**.

It supports publishing files from:

- Local folders
- External HDD / USB drives
- Mapped network drives
- NAS / UNC paths

It also supports:

- Server IP and port configuration
- Adding and removing storage paths
- Firewall rule creation
- Docker-based deployment
- Full cleanup options

---

# Main Features

- Install / Reconfigure
- Start / Stop
- Add Storage Path
- Delete Storage Path
- IP / Port Setup
- Show Compose File
- Docker Cleanup
- Full Uninstall
